#  CustOS Portfolio

CustOS Portfolio é um **Web OS interativo** desenvolvido como portfólio pessoal, que simula um sistema operacional rodando diretamente no navegador.

Cada “usuário” do sistema representa uma **área profissional em que atuo**, permitindo explorar habilidades, conceitos e projetos de forma visual, modular e imersiva.

---

## Conceito do Projeto

O CustOS simula **4 usuários/perfis**, cada um focado em uma área específica de atuação:

- **Fullstack & Mobile Developer**
- **DevOps**
- **Bitcoin Developer**
- **Data Science**

Cada perfil funciona como um ambiente independente dentro do Web OS, com seus próprios componentes, interfaces e fluxos.

---

## Tecnologias Utilizadas

- **Next.js (App Router)**
- **React**
- **TypeScript**
- **GSAP**
- **Framer Motion**
- **Swiper**
- **CSS / UI Componentizada**
- Arquitetura baseada em **features e componentes reutilizáveis**

##  Como rodar o projeto localmente

```bash

# ou
yarn install
# ou
pnpm install

# ou
```
